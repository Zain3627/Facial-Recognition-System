# Facial Recognition System

This is a Streamlit application for an end-to-end facial recognition system using FaceNet.

The application allows for:
- **Face Identification** from uploaded images, camera snapshots, or live video.
- **User Registration** via a guided 5-shot webcam capture process.

This Space is deployed from the [Zain3627/Facial-Recognition-System](https://github.com/Zain3627/Facial-Recognition-System) GitHub repository.
